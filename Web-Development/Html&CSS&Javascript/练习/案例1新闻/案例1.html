中央气象台发布台风蓝色预警
“桑达”给华东沿海带来风雨影响

【字体：大中小】打印本页分享：发布时间：2022年07月30日 来源：中国气象报社

“桑达”未来强度变化不大

今年第5号台风“桑达”（热带风暴级）的中心今天（7月30日）上午8点位于江苏启东偏东方向大约450公里的东海北部海面上，中心附近最大风力有8级（18米/秒），中心最低气压1000百帕，七级风圈半径120~180公里。中央气象台7月30日10时发布台风蓝色预警。

中央气象台台风与海洋气象预报中心首席预报员王海平表示，预计“桑达”将以每小时20公里左右的速度向西偏北方向移动，强度变化不大，将于31日移速减慢，在江苏近海北上，由东海北部进入黄海海域，强度逐渐减弱。

目前，“桑达”呈东西向狭长带结构，水平尺度较小，后期强度可能会有所发展，但强度并不会很强，预计中心附近最大风速能达9级、阵风10级。

“桑达”给华东沿海带来风雨影响

王海平分析，当前“桑达”南侧有另一热带扰动活动，两者合并可能性较大，合并后“桑达”强度可能略有增强。尽管如此，“桑达”整体强度弱、影响时间短、影响范围小，未来并不会登陆，对我国影响不大，但会给华东沿海部分地区带来风雨影响。

受“桑达”影响，7月30日至31日，东海北部、黄海中部和南部、长江口区、浙江东北部沿海、上海沿海、江苏南部沿海将有6~7级风，“桑达”中心经过的附近海面风力有8~9级，阵风可达10级。7月30日至31日，江苏南部沿海、浙江北部沿海、上海地区将有中到大雨，局地暴雨。

受影响地区注意防范强降水可能引发的山洪、地质灾害，公众应留在安全处避风，危房内的人员要及时转移。

　

另一方面，“桑达”给华东沿海地区带来的降水将暂时缓解近期的高温和干旱。王海平表示，后期，副热带高压再度西伸加强后，还会给华东区域带来高温天气。

今年7月份台风较往年同期偏少

根据1949年到2021年台风数据，截至目前我国7月仅生成两个台风，相较历史同期平均偏少2个。这是由于今年大范围的副热带高压长时间控制西太平洋地区，难以形成对流，无法达到台风生成所需的必要条件。

王海平表示， 8月上旬在菲律宾以东洋面以及我国南海海域还将会有1~2个台风生成，需持续关注对我国的影响。
